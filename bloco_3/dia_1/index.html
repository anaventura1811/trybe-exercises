Sobre a dupla da Dinâmica do GitHub

Esta dupla foi formada com o objetivo de entender alguns conceitos-chave super úteis no
dia-a-dia de qualquer pessoa desenvolvedora: trabalho colaborativo num mesmo projeto,
mediante o compartilhamento de versões entre as pessoas do time até a consecução completa
do trabalho. Entendemos na prática a importância de saber utilizar de forma correta o Git
e o GitHub e o quanto estas ferramentas tornaram mais fácil e ágil o desenvolvimento de
projetos em grupo - sejam eles grandes ou pequenos. De quebra, ainda deu para aprender
bastante sobre ferramentas complementares de estudo!
