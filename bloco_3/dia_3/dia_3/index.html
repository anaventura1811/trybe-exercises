Lista dos Melhores Sites que já vi

Top 3 dos melhores sites que existem na web

  1. Para ler artigos interessantes sobre quase todos os temas:Medium
  2. Para encontrar receitas que funcionam:Panelinha
  3. Para buscar referências e inspirações de design:Pinterest

Principais motivos para eu gostar deles

  • O primeiro site possui uma grande variedade de artigos em idiomas variados
  • O segundo site, além de ser eclético, é também fácil de mexer e possui receitas com passo-a-passo bem explicado
  • Por fim, o terceiro site é excelente para uma pesquisa rápida de referências visuais e possibilita que se crie pastas para cada tema/referência pesquisado. Isso é muito útil para projetos.
